# Tarea-2_Threads_Carrera

## Requisitos
- JDK 11 o superior
- VS Code con la extensión **Extension Pack for Java**

## Desde VS Code
1. Abrir la carpeta `Threads_Carrera/` en VS Code (**Archivo → Abrir carpeta**)
2. Abrir `Main.java`
3. Presionar **▷ Run** (esquina superior derecha)

## Desde terminal
```bash
# Compilar
javac -d out Main.java model/*.java view/*.java controller/*.java

# Ejecutar
java -cp out Main
```
